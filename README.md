
# Watch Shop

Responsive Web Application with Authentication, Search, and API Integration


## Overview

This project is a fully responsive web application designed to deliver a seamless experience across devices. It includes user authentication, profile management, data persistence using local storage, and dynamic content powered by an external API. The app supports both light and dark modes, ensuring accessibility and an enjoyable user experience.

 

## Features Implemented

### 1. Responsiveness

* The entire layout adapts to desktop, tablet, and mobile screens.
* Utilizes flexbox and CSS grid for scalable and flexible layouts.
* Verified through manual testing on multiple device sizes and orientations.

 

### 2. Hosting

* The project is hosted on GitHub Pages for public access.
* Repository includes a detailed README.md describing setup, features, and deployment.

Live Demo: https://amazing-youtiao-a7e4d8.netlify.app/
 

### 3. Light and Dark Modes

* Implemented theme toggling using JavaScript and CSS variables.
* User’s theme preference is saved in localStorage for persistence.
* All pages automatically load the last selected mode.

 

### 4. Design Quality

* Clean, professional, and minimal UI design.
* Consistent color palette ensuring readability and contrast in both modes.
* Responsive typography and properly aligned content.
* Every visual element has a clear functional purpose (no placeholders or non-functional components).

 

### 5. Enhanced JavaScript Functionality

#### Authentication (Local Storage Based)
* Sign Up and Log In implemented with form validation.
* On successful login, the user is redirected to a Profile Page displaying their saved data (name, email, etc.).
* User sessions are managed through localStorage for persistent login state.

#### Rating System
* Users can rate items, and ratings are saved to localStorage for later access.

#### Form Validation
* Comprehensive validation for:
  * Email format
  * Password length and complexity
  * Phone number format
  * Required fields
* Real-time error messages and input highlights for invalid data.

#### Search and Filter Features
* Users can search and filter content dynamically.
* Results are stored in localStorage, allowing users to revisit recent searches without re-entering data.
* No placeholder or inactive buttons — all interface elements are fully functional.



### 6. External API Integration
* Integrated with a relevant external API (e.g., OMDb API for movies or Google Books API for books).
* Fetches and displays live data dynamically.
* Results are updated in real time and can be rated or saved by users.



### 7. Feature Cohesion and Thematic Relevance

* All features (authentication, rating, search, API data) are directly related to the app’s central theme.
* The design and functionality are aligned to provide an intuitive, unified user experience.



### 8. Presentation and Delivery

* Clear, concise explanation of all features during the presentation.
* Smooth navigation demo across pages, showcasing responsiveness, authentication, and API integration.
* Strict adherence to time limits and professional presentation flow.
